# code-snapshot

Turn your code into beautiful shareable images.

Live at: **https://devkoan.github.io/code-snapshot/**

## Features

- Syntax highlighting — Swift, Python, JS, TS, Kotlin, Java, Go, Rust, CSS, Bash
- Line highlighting with custom color
- Diff view (added / deleted lines)
- Callout annotation bubbles
- 4 themes — Dark, Light, Gradient, Ocean
- macOS / Minimal / No window chrome
- Adjustable padding, font size, preview width
- Copy PNG / Save PNG (3x resolution)
- devkoan watermark toggle

## Deploy

Single `index.html` — no build step.

1. Clone this repo
2. Settings → Pages → Source: `main` branch, `/ (root)`
3. Live at `https://devkoan.github.io/code-snapshot/`
