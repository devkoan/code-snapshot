About · Code Snapshot

코드를 아름다운 이미지로.

Code Snapshot은 소스 코드를 공유하기 좋은 이미지로 변환해주는 무료 웹 도구입니다. 블로그, 발표 자료, SNS, PR 리뷰 — 코드를 스크린샷으로 남겨야 할 모든 순간에 쓰세요. 계정도, 설치도, 업로드도 필요 없습니다.

왜 만들었나

개발자는 하루에도 몇 번씩 코드를 이미지로 공유합니다. 하지만 대부분의 도구는 가입이 필요하거나, 서버로 코드를 업로드하거나, 원하는 하이라이트·주석 기능이 부족합니다. Code Snapshot은 단일 HTML 파일로 브라우저 안에서 전부 동작합니다 — 코드는 절대 디바이스를 떠나지 않습니다.

주요 기능

Syntax Highlight
Swift, Python, JS/TS, Kotlin, Java, Go, Rust, CSS, Bash 지원.
Line Highlight
특정 라인을 지정 색상으로 강조. 포인트 전달이 쉬워집니다.
Diff View
추가/삭제된 라인을 한눈에. 리뷰나 before/after 공유에 적합.
Callout 주석
코드 위에 말풍선 형태의 설명을 달 수 있습니다.
4가지 테마
Dark, Light, Gradient, Ocean — 상황에 맞게 선택.
윈도우 스타일
macOS / Minimal / None — 원하는 창 프레임을 골라 쓸 수 있습니다.
세밀한 레이아웃
패딩, 폰트 크기, 프리뷰 너비를 자유롭게 조절.
3x PNG 내보내기
고해상도 PNG로 저장하거나 클립보드로 바로 복사.

사용 방법

  1. 홈 화면에서 코드를 붙여 넣거나 예시 코드를 수정합니다.
  2. 좌측 패널에서 언어, 테마, 윈도우 스타일, 패딩을 설정합니다.
  3. 필요하면 라인 하이라이트·diff·콜아웃을 추가합니다.
  4. Copy PNG 또는 Save PNG 버튼으로 내보냅니다.

프라이버시

모든 렌더링은 브라우저에서 일어납니다. 서버로 코드를 보내지 않고, 계정이나 쿠키 기반 추적도 사용하지 않습니다. 폐쇄망 환경에서도 외부 폰트·스크립트를 캐싱한 뒤 오프라인으로 쓸 수 있습니다.

오픈소스

Code Snapshot은 단일 index.html로 빌드 단계가 없습니다. GitHub에서 소스를 확인하거나 포크해서 자신의 스타일로 커스터마이즈할 수 있습니다.
