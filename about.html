About · Code Snapshot

Turn your code into beautiful images.

Code Snapshot is a free web tool that converts source code into clean, shareable images. Use it for blog posts, slide decks, social media, PR reviews — anywhere you need to screenshot code. No account, no install, no upload.

Why we built it

Developers share code as images several times a day. But most tools require sign-ups, upload your code to a server, or lack the highlighting and annotation features you actually need. Code Snapshot is a single HTML file that runs entirely in your browser — your code never leaves your device.

Features

Syntax Highlighting
Swift, Python, JS/TS, Kotlin, Java, Go, Rust, CSS, and Bash.
Line Highlight
Highlight specific lines in a custom color to draw attention.
Diff View
Show added and deleted lines — perfect for reviews and before/after.
Callout Bubbles
Add speech-bubble annotations on top of your code.
4 Themes
Dark, Light, Gradient, Ocean — pick the one that fits.
Window Styles
macOS, Minimal, or None — choose the window chrome you prefer.
Fine-grained Layout
Adjust padding, font size, and preview width to your taste.
3× PNG Export
Save as high-resolution PNG or copy straight to your clipboard.

How to use

  1. Paste your code on the home screen, or edit the sample code.
  2. Pick language, theme, window style, and padding from the left panel.
  3. Optionally add line highlights, diff marks, or callout bubbles.
  4. Hit Copy PNG or Save PNG to export.

Privacy

Everything renders in your browser. We don't send your code to a server, and there's no account or cookie-based tracking. Once the fonts and scripts are cached, the whole tool works offline — handy for air-gapped environments.

Open source

Code Snapshot is a single index.html with no build step. Browse the source on GitHub or fork it and make it your own.
